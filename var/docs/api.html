HTTP status code 200

Liefert die angefragten Informationen zur Fahrt, wie sie im System vorliegen (soweit vorhanden) zurück. Zum jetzigen Zeitpunkt können hier bereits Informationen zum voraussichtlichen Ankunftszeitpunkt, zum zugewiesenen Fahrer bzw. dem Fahrzeug und dessen Position vorliegen.

Body

Type: application/json

Example:

{
  "id": "abf0123ff",
  "status": "1",
  "eta": "2015-07-06T13:45:00.719Z",
  "driver":{
    "id": "8x8_Klaus",
    "phoneNumber": "017212345678"
  },
  "car" {
    "id": "8x8_DD-XY-1",
    "phoneNumber": "017212345678",
    "longitude": 13.741274,
    "latitude": 51.060584
  }
}
